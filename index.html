Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and reder htem into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the apearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered pages. It providesc a means to create structured documents by denoting stuctural semantics for text such as headings, paragraphs, list, links, quotes and other items. HTML elemetns are delineated by tags, written using angle brackets.
